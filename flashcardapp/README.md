# codealpha__flashcardapp
**Android Development Internship – CodeAlpha**

## Project Description
This is a simple and interactive **Flashcard Android Application** developed as part of the
CodeAlpha Android Development Internship.  
The app helps users to learn and revise concepts easily using question–answer flashcards.

## Features
- Add new flashcards
- View flashcards
- Simple and clean user interface
- Easy navigation
- Beginner-friendly design

## Tools & Technologies Used
- Java
- Android Studio
- XML
- Gradle

## How to Run the App
1. Clone the repository https://github.com/dharshinimathiyazhakan/codealpha__flashcardapp.git
2. Open the project in **Android Studio**
3. Let Gradle sync complete
4. Run the app using an **Emulator** or **Physical Android Device**
